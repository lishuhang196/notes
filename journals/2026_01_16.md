- Git远程分支删除后，`git branch -a`后仍然会显示删除分支 #git
	- `git fetch --prune` 将远程没有，本地有的分支删除。
	- `git remote prune origin` 删除本地分支存在但远程分支已不存在的分支。
	- `git branch -d/D dev` 删除本地分支，`-D` 强制删除。
	- `git push origin --delete <branch>` 删除远程分支.
