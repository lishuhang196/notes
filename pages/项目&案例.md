- 前端如何实现网页截图
  id:: 695df815-bb2d-4cda-9c11-30783dbef11b
	- 案例：[Render DOM to Canvas — Example](https://codepen.io/bigfox563/pen/PoMqvXE)
	- 核心原理：
		- 处理`DOM`元素，将元素的样式添加到`Style`属性中
		  logseq.order-list-type:: number
		- `XMLSerializer`对象将`DOM`元素序列化
		  logseq.order-list-type:: number
		- 将`XML`字符串插入到`svg`中`foreignObject`标签中
		  logseq.order-list-type:: number
		- 使用svg创建图片
		  logseq.order-list-type:: number
		  ```javascript
		  const svgString = "..."
		  const blob = new Blob([svgString], { type: "image/svg+xml;charset=utf-8" })
		  const url = URL.createObjectURL(blob)
		  const img = new Image()
		  img.src = url
		  ```
